## Installing the Chart

### Helm repository

```bash
# Add the repository
helm repo add tillo-library https://TilloTech.github.io/tillo-platform-helm-library/

# Install the chart
helm install tillo-library app-template -f values.yaml
```

### OCI

```bash
# Install the chart
helm install oci://ghcr.io/TilloTech/helm/app-template -f values.yaml
```
